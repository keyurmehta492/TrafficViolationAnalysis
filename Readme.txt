Project: Analysis on Traffic Violation in the US

Steps to launch project:
1. Open below mentioned url in any browser.
http://iupui-big-data-project-final.s3-website.us-east-2.amazonaws.com/

2. Select category and click on 'Show Map' button to launch the visualization.
3. Select desired type from the dropdown and click on 'Chart' button to launch the visualization.
4. More Analyses which shows visualization from Amazon Quicksight.
